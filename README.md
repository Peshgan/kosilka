# Косилка.by

Лендинг услуг по покосу травы, газона и бурьяна в Минске и Минской области.

## Стек
Чистый **HTML / CSS / JS** без фреймворков. Один файл — быстрый рендер, лёгкий деплой.

## Файлы
- `index.html` — основной сайт: интерактивный canvas-герой (blob «до/после» под курсором), счётчики, карточки услуг, цены, шаги, отзывы, FAQ, форма заявки.
- `profpokos.html` — вариант лендинга в классическом светлом стиле.
- `images/` — фотографии услуг и изображения для эффекта «до/после».
- `pp-assets/` — дополнительные изображения.

## Запуск локально
Любой статический сервер, например:

```bash
npx http-server -p 8080
# затем открыть http://localhost:8080
```

## Деплой
Статика — можно хостить на GitHub Pages, Vercel, Netlify или любом веб-сервере.

## Особенности
- Адаптивность (mobile-first), без горизонтального скролла на 390–1440px
- Интерактивный hero на Canvas 2D (DPR-aware, blob-reveal по курсору и тачу)
- Scroll-анимации, анимированные счётчики, spotlight на карточках
- `prefers-reduced-motion` уважается
